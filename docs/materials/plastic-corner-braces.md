# Plastic Corner Braces

Lightweight 90° plastic brackets used to lock plywood triangles together and keep the wedge angle stable.

## Overview
- Plastic avoids marring the acrylic surface and is easy to bond with super glue.
- Wide enough flange to spread load on 1.5 mm plywood triangles without splitting.
- Reusable if cleaned of cured glue; keep a few spares to replace any clogged with epoxy.

## Usage notes
- Roughen the bonding faces lightly with sandpaper before gluing for better adhesion.
- Check that brace height clears the acrylic sheet edges so the wedge sits flat.
